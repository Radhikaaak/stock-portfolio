# Stock Portfolio Tracker

## Description
A simple Python project that allows users to enter stock names and quantities, calculate the total investment value using predefined stock prices, and generate a portfolio summary.

## Features
- Enter stock names and quantities
- Calculate total investment value
- Uses predefined stock prices
- Displays portfolio summary
- Saves results to a text file

## Technologies Used
- Python
- Dictionary
- Loops
- User Input/Output
- File Handling

## How to Run
1. Download the project files.
2. Open the project in VS Code.
3. Run:

```bash
python stock_portfolio.py
